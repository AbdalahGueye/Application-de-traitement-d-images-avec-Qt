# Application-de-traitement-d-images-avec-Qt
(PROJET A BUT PEDAGOGIQUE)
Un logiciel modulaire permettant d’acquérir une image (webcam ou fichier), d’enchaîner plusieurs traitements d’image OpenCV, de visualiser chaque étape, puis de sauvegarder le résultat final. L’interface graphique permet d’ajouter, paramétrer ou supprimer interactivement les traitements sous forme de “boîtes” connectées dans un pipeline visuel.
